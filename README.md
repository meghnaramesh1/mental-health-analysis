# Mental Health Analysis Project

## 📌 Project Overview
This project analyzes mental health survey data to identify patterns and factors affecting mental well-being. The goal is to clean, preprocess, and build a machine learning model to understand mental health trends.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Scikit-learn

---

## 📊 Key Features
- Data cleaning and preprocessing
- Handling missing values and duplicates
- Feature transformation (date, age, gender standardization)
- Machine learning model using Random Forest Classifier
- Model evaluation using accuracy and classification report

---

## 📈 Machine Learning Model
- Random Forest Classifier (for mental health prediction)

---

## 📊 Outputs
- Cleaned dataset
- Model predictions
- Accuracy score
- Classification report

---

## 🚀 How to Run the Project
1. Install required libraries:
   ```bash
   pip install pandas scikit-learn
   ```

2. Run the Python script:
   ```bash
   python "Mental Health Performance.py"
   ```

---

## 📌 Key Insights
- Identifies patterns in mental health survey responses
- Helps understand demographic impact (age, gender)
- Useful for basic mental health trend analysis
